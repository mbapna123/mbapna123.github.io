For this project, a user enters their height as well as the name or the ID number of a particular pokemon. The app will retrieve a JSON file from the PokeAPI and parse the JSON file, finding the height and an image file of the Pokemon. Based off of the proportion of the user and the pokemonâ€™s height, a scale illustration will be shown of the user and the pokemon. To use this project, one might have to enable this Google Chrome Extension (CORS) as well as potentially load unsafe scripts.

CORS is Cross-Origin Resource sharing. Essentially, for security reasons, many servers only allow XMLHTTPREQUESTS from the same domain. For example, only in a website that has the domain of www.mayankGOAT.com can access the JSON file www.mayankGOAT.com/mayankBOSS.json. But cross origin resource sharing would allow you access this JSON file from any domain.
