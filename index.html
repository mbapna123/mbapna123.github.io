For this project, a user enters their height as well as the name or the ID number of a particular pokemon. The app will retrieve a JSON file from the PokeAPI and parse the JSON file, finding the height and an image file of the Pokemon. Based off of the proportion of the user and the pokemon's height, a scale illustration will be shown of the user and the pokemon.

This website was designed using JQuery, JQueryUI, and AngularJS. No external templates were used.
